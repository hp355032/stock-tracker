🔗 分享給朋友使用

這個網站每個人都能用，但每個人需要建立自己的 Google Sheets 來存資料。
把以下步驟傳給你的朋友就好：

建立自己的 Google 試算表

到 sheets.google.com 建一份新的空白試算表

建立 Apps Script 後端

在試算表裡點「擴充功能 → Apps Script」，把程式碼貼上去。
程式碼可以從你這邊拿，或到 GitHub repo 下載 google-apps-script.js

部署 Apps Script

點「部署 → 新增部署作業」，類型選「網頁應用程式」，
執行身分選「我」，存取權限選「所有人」，然後按部署

複製部署網址

部署成功後會得到一串 https://script.google.com/macros/s/xxx/exec 網址

打開網站，貼上網址

把你的網站網址傳給朋友，他打開後在上方貼上自己的 Apps Script 網址，點「連結」就完成了！

💡 重點：

網站是同一個，但每個人連結的是自己的 Google Sheets。
你的資料只有你看得到，朋友的資料只有朋友看得到，互不干擾。
